Quiz Master

This application is developed for educational purpose, allowing the users to prepare the multiple choice questions for different examinations conducted on any organizational level. The main goal of the application is to enable users to practice for objective tests conducted for admissions and recruitment, with focusing on different fields.

This Web Application provides facility to conduct online examination worldwide. It saves time as it allows number of students to give the exam at a time and displays the results as the test gets over, so no need to wait for the result. It is automatically generated by the server. Administrator has a privilege to create, modify and delete the test papers and its particular questions. User can register, login and give the test with his specific id, and can see the results as well.

Quiz Master

This quiz application includes five topics to choose from , namely (i) Mathematics, (ii) Physics, (iii) Chemistry (iv) Computer Science (v) Networking

It will help different people in learning and answering different questions created by other users after choosing particular category. It also has a facility for user login where user can login and can take quiz of his choice. He can take as many quizzes he wants and get the score instantly.
